# Toshan Portfolio

Dynamic and immersive 3D portfolio built with React, Three.js, and Framer Motion.

## 🚀 Features

- Interactive 3D Elements
- Smooth Animations
- Responsive Design
- Glassmorphism UI

## 🛠️ Tech Stack

- **Frontend**: React.js, Vite
- **3D Graphics**: Three.js, React Three Fiber
- **Animations**: GSAP, Framer Motion
- **Styling**: Tailwind CSS
